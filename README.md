# Michael Adkins | Professional Saxophonist & Educator

This is the official digital portfolio and pedagogy archive for **Michael Adkins, DMA**. 

### Overview
A minimalist, high-impact web presence designed with a "less is more" intentionality, reflecting an artistic philosophy rooted in the creative hubs of **New York and Boston**.

### Site Structure
* **Home**: Core metrics and critical acclaim.
* **About**: Detailed biography and sublimated musical influences.
* **Discography**: A definitive archive of recorded works, including releases on **HatHut Records**.
* **Pedagogy**: Doctoral-level woodwind instruction and studio information.
* **Contact**: Direct gateway for performance and recording inquiries.
